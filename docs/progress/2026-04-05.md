- `usbipd-win 5.3.0` を導入した
- `usbipd bind --busid 2-3` に管理者権限が必要であることを確認した

## 判明したこと

- TL866II Plus を WSL2 へ渡すには、Windows 側の `usbipd bind` を管理者 PowerShell で先に実行する必要がある
- その後に通常権限の PowerShell で `usbipd attach --wsl --busid ...` を実行できる
- WSL 側で `minipro` を試すには Ubuntu に `minipro` と `usbutils` を入れる必要がある
- この確認環境では、Windows 側管理者権限と Ubuntu 側 `sudo` がないと最後まで自動実行できない
