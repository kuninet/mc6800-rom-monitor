# KiCad 回路図

このディレクトリには、MC6800 ROM モニタ向けのアドレスデコード検討用回路図を置きます。

## 現在の回路図

- `acia_decode_fcf4_fcf7.kicad_sch`
  - KiCad 9 互換
  - `74LS138`、`74LS688`、`74LS00` を使った `FCF4-FCF7` 緩和版の ACIA デコード案
  - 実部品の厳密なピン配置ではなく、信号名と論理構成を整理するためのブロック図

## 図面の意図

- ROM は `E000-FFFF` を担当
- ACIA は `FCF4-FCF7` をデコード窓とする
- `FCF4/FCF5` を主使用アドレスとし、`FCF6/FCF7` はミラー扱い
- ACIA 選択時は ROM を無効化する
